點擊工具列圖示後，**直接在目前分頁上覆蓋一個全屏介面**，無需另開新分頁。再點一次圖示或右上角 × 即可隱藏，回到原本頁面。

## 1. 功能介紹

### 搜尋

| 功能 | 說明 |
|------|------|
| 日期選擇 | 自訂月曆下拉元件；週六藍字、週日紅字；國定假日橘色底色加圓點標示 |
| 日期快捷鍵 | **今天**（重置為今天）、**+1D**、**+1W**、**+2W**（以目前選取日期為基準累加） |
| 時段選擇 | **上午**、**下午**、**全天**（依序查詢上午＋下午後合併） |
| 大樓選擇 | 松仁、仁愛、敦化 |
| 偏好記憶 | 搜尋條件自動儲存，下次開啟自動還原；過期日期自動換為今天 |

### 篩選

篩選即時套用，不重新呼叫查詢。

| 篩選項目 | 說明 |
|----------|------|
| 起始時間 | 依搜尋結果動態產生，需搭配「時間長度」確認連續可預訂區間 |
| 時間長度 | 0.5 hr / 1 hr / 1.5 hr / 2 hr |
| 容納人數 | 不限 / 5+ / 10+ / 15+ / 20+ |
| 樓層多選 | 依搜尋結果動態產生，可多選 |
| 隱藏不常用會議室 | 過濾名稱包含「階梯教室」「中心樹」的會議室 |
| 排除不常用時間 | 隱藏 07:00–09:00、12:00–13:30、18:00 後的時段格 |

### 檢視模式

| 模式 | 說明 |
|------|------|
| 牌卡（Card） | 每間會議室一張卡片，列出空檔時段與已預訂資訊 |
| 時間表（Table） | 橫軸為時間、縱軸為會議室；已預訂格自動合併顯示 |

### 其他

- **最愛會議室**：點擊 ★ 加入最愛，最愛的會議室排列在最前方
- **會議室詳情**：點擊會議室名稱可查看容量、設備、圖片輪播（支援放大）
- **預訂確認**：點擊空檔後顯示確認視窗，確認後跳轉至預訂頁面
- **國定假日標示**：支援載入內建假日或匯入 CSV，月曆上以橘色標示
- **登入狀態**：Header 顯示登入狀態，未登入時提供快速前往登入的按鈕

## 2. 安裝方式

1. 開啟 Chrome，前往 `chrome://extensions/`
2. 開啟右上角「**開發者模式**」（Developer mode）
3. 點擊「**載入未封裝項目**」（Load unpacked）
4. 選擇本目錄（`cb-overlay/`）
5. 工具列出現「CB 會議室」圖示即安裝完成

---

## 3. 使用說明

**前置條件：必須先在會議室預訂系統完成登入，並保持該分頁開啟。**

```
1. 前往會議室預訂系統並登入
2. 點擊工具列「CB 會議室」圖示 → 覆蓋介面開啟
3. Header 顯示「已登入」（綠色）
4. 選擇日期、時段、大樓 → 點擊「搜尋空房」
5. 使用篩選面板縮小範圍
6. 點擊空檔時段 → 確認視窗 → 「前往預訂」→ 在新分頁完成送出
7. 再點一次圖示或按右上角 × 關閉覆蓋介面
```

---

## 4. 注意事項

- 若顯示「站台頁面未就緒」，請**重新整理**預訂系統頁面後再試
- 若顯示「Session 已過期」，請重新登入後點擊「重新檢查」
- 全天查詢會依序發出兩次請求（上午 → 下午），時間稍長屬正常現象
- 本 Extension 僅限內部使用，不上架 Chrome Web Store

## 5. 檔案結構

```
cb-overlay/
├── manifest.json              # MV3 設定檔
├── background.js              # Service Worker：點擊 icon 切換 overlay
├── content-bridge.js          # Content Script：注入頁面，橋接 cookie / CSRF
│
├── html/
│   └── app.html               # Overlay iframe 入口 HTML
│
├── js/
│   ├── api.js                 # API 封裝（postMessage RPC）
│   ├── parser.js              # HTML → 結構化資料
│   └── app.js                 # UI 狀態、渲染、篩選、Modal、假日邏輯
│
├── css/
│   └── app.css                # UI 樣式
│
├── assets/
│   └── 115_tw_holidays.csv    # 民國 115 年國定假日（政府資料開放平台格式）
│
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```
